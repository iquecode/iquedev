BackEnd

Criação de código em PHP/JS para a equipe de desenvolvedores do site uoldown.com, o que resolveu o problema de gargalo de uso de memória no servidor Apache com relação às requisições de Downloads dos arquivos.
Usando conceitos avançados de buffer data, a classe criada mudou o comportamento de envio dos dados, com um controle de envio por partes, carregando e descarregando da memória durante o envio, aumentando em 50x a capacidade de envio para o mesmo hardware.
Implementado também integração completa com o sistema do cliente e autenticação por Token para aumento da segurança.
